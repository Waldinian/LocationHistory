Dependencies:
  jq
  
Python3 dependencies:
  matplotlib
  numpy
  pandas
  
To use:
  0) download your google location history data as a .json archive using takeout.google.com, place the unzipped directory in this directory
  1) Change the lines in Location_History_TakeoutJSON_to_CSV.sh that tell you to do so
  2) Run the script Location_History_TakeoutJSON_to_CSV.sh
    *) To change the output image resolution, edit, the second-to-last line
       
  3) Wait a while
